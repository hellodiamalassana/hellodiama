# marcomonzon

My own website made with LitElement!

## Install dependencies
```bash
npm i
```

## Serve project
```bash
npm start
```

## Testing using karma
```bash
npm run test
```

## Eslint scan
```bash
npm run lint
```

## Transpile with Babel inside /lib
```bash
npm run build
```

## Pack everything for deployment
```bash
npm run bundle
```

## Deploy up to gh-pages
```bash
npm run deploy
```
